Task Tracker Version 0.0.1

Task Tracker helps users track large numbers of documents over a variety of users, tasks and responses.

To be able to track tasks, each task must be entered into the database via the File -> New Task menu item.

Tasks in the database are displayed in a table in the main frame under the search box

To add a response to a task, right click the task's row in the task table and a popup menu will appear giving various options. Select Add Response

To view a task and its responses, right click the task's rows in the task table and a popup menu will appear giving various options. Select View
